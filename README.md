# euler

A repository for collecting work on Project Euler problems.

## Installation

    $ git clone https://github.com/emdeesee/euler.git

## Usage

    $ lein run [number]

## Examples

    $ lein run 1
    "Elapsed time: 30.94175 msecs"
    233168
